Learning Path

Understand Core Entities — Query PATIENTS, ORGANIZATIONS, PROVIDERS, PAYERS to understand the actors in the system.
Explore Encounter Types — Analyze ENCOUNTERS by class (ambulatory, emergency, inpatient, wellness, urgentcare) and cost patterns.
Clinical Deep Dive — Join ENCOUNTERS to CONDITIONS, PROCEDURES, MEDICATIONS to understand what happens during visits.
Financial Analysis — Trace money flow: ENCOUNTERS → CLAIMS → CLAIMS_TX (charges, payments, adjustments, transfers).
Population Health — Aggregate conditions by demographics, analyze payer coverage vs out-of-pocket costs.
Advanced Analytics — Time-series on encounters, provider utilization, medication patterns, imaging trends.

Sample Queries

-- Top 10 most common conditions SELECT DESCRIPTION, COUNT(*) AS frequency FROM SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.CONDITIONS GROUP BY DESCRIPTION ORDER BY frequency DESC LIMIT 10;
-- Average encounter cost by class SELECT ENCOUNTERCLASS, AVG(TOTAL_CLAIM_COST) AS avg_cost, COUNT(*) AS encounter_count FROM SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.ENCOUNTERS GROUP BY ENCOUNTERCLASS ORDER BY avg_cost DESC;
-- Full patient journey (multi-table join) SELECT p.FIRST || ' ' || p.LAST AS patient, e.ENCOUNTERCLASS, e.ENCOUNTER_START, c.DESCRIPTION AS condition, med.DESCRIPTION AS medication, pay.NAME AS insurer FROM SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.PATIENTS p JOIN SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.ENCOUNTERS e ON p.PATIENT_ID = e.PATIENT_ID LEFT JOIN SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.CONDITIONS c ON e.ENCOUNTER_ID = c.ENCOUNTER_ID LEFT JOIN SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.MEDICATIONS med ON e.ENCOUNTER_ID = med.ENCOUNTER_ID JOIN SYNTHETIC_HEALTHCARE_DATA__CLINICAL_AND_CLAIMS.SILVER.PAYERS pay ON e.PAYER_ID = pay.PAYER_ID LIMIT 20;
